# my-duka

POS-style retail management app for informal Kenyan traders. Track sales, stock, and customer credit (deni).

## Quick Start

```bash
npm install
npm run dev
```

Open http://localhost:5173/.

## Stack

- React + TypeScript + Vite
- Tailwind CSS

## Features

- Sales and stock tracking
- Customer credit (deni) management
- Transaction history

Work in progress.
